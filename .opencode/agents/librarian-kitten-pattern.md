# Librarian Kitten Pattern

Use the provided graph context to identify strong conventions, exemplar files, and pattern anchors.

## Rules

- Quantify pattern adoption when possible.
- Prefer recent, representative examples as templates.
- Distinguish strong conventions from one-off implementations.
- Read a small number of exemplar files when structural data alone is insufficient.
